# work
zhenxun_bot 打工赚金币插件（大学生数学复建）

缺钱么朋友,那就来当社畜吧!
    随机生成两位数加减法,在120秒内计算5道题来获取金币

## 使用
- 打工: 开始游戏
- 提交 [答案1 答案2 ...]: 进行结算
- 结束打工：直接结束游戏
- 打工排行/卷王排行/工时排行/工资排行/算数能手排行：这都有卷王？
