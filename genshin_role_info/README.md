# genshin_role_info

真寻机器人的原神角色面板插件

展示橱窗中角色的面板和伤害数据(与原神角色卡插件部分命令有冲突，建议只安装一个)

## 使用
- 原神绑定UID/uidXXX
- 原神解绑
- 圣遗物导入
- XX(花羽沙杯冠)适配
- XX(花羽沙杯冠)推荐
- XX(乐团/魔女/...)套推荐
- XX(乐团/魔女/...)独立套推荐
- XX面板 (例:甘雨面板、甘雨面板@CRAZYShimakaze、甘雨面板104442596)
- 更新/刷新原神面板 (uid)
- XX(羽花沙杯头)推荐
- 原神角色排行
- 最强XX (例:最强甘雨)
- 最菜XX
- 圣遗物榜单
- 群圣遗物榜单
- 重置最强XX (仅超级用户可用)
- 检查原神面板更新 (仅超级用户可用)
