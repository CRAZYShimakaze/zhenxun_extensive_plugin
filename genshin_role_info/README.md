# genshin_role_info

真寻机器人的原神角色面板插件

展示橱窗中角色的面板和伤害数据(与原神角色卡插件部分命令有冲突，建议只安装一个)

## 使用

- 原神角色卡 uid 角色名
- 更新角色卡 uid
- XX面板 (例:甘雨面板、甘雨面板@CRAZYShimakaze)
- 更新面板
- 我的角色
- 他的角色 @CRAZYShimakaze
- 最强XX (例:群最强甘雨)
- 圣遗物榜单
- 群圣遗物榜单
- 重置最强XX (仅超级用户可用)
- 检查面板更新 (仅超级用户可用)
