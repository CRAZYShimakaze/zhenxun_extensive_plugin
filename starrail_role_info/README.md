# starrail_role_info

真寻机器人的星铁角色面板插件

展示漫游签证中角色的面板和伤害数据

## 使用

- XX面板 (例:希儿面板、希儿面板@CRAZYShimakaze、希儿面板104442596)
- 更新/刷新星铁面板 (uid)
- XX(头手身脚球绳)推荐
- 我的星铁角色
- 他的星铁角色@CRAZYShimakaze
- 最强XX (例:最强希儿)
- 最菜XX
- 遗器榜单
- 群遗器榜单
- 重置最强XX (仅超级用户可用)
- 检查星铁面板更新 (仅超级用户可用)
