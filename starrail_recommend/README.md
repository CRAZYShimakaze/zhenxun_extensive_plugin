# starrail_recommend

zhenxun_bot 星铁攻略

展示星铁攻略

## 使用

- 光锥评级/推荐/建议
- 遗器推荐/适配/评级
- 角色配队/阵容
- XX攻略
- XX图鉴
- XX素材/XX材料
- 更新星铁推荐 (仅超级用户可用)
- 检查星铁插件更新 (仅超级用户可用)
